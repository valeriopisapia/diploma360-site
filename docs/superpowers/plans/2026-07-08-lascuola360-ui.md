# La Scuola360 divergent UI — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add La Scuola360's divergent UI (restructured nav, umbrella Home, unified Chi siamo, a Ripetizioni section of 4 pages + form) as a brand-gated superset, leaving Diploma360 byte-for-byte unchanged.

**Architecture:** Everything keys off `brand.id` (build-time). Shared routes (`/`, `/chi-siamo`) become thin selectors that render a per-brand component. La-Scuola360-only routes (`/ripetizioni/*`) are gated with `notFound()`. Nav becomes a data-driven per-brand model; the footer is chosen per-brand in `ChromeGate`. Ripetizioni leads reuse `/api/lead` with a `PRODOTTO` marker on Brevo list 41.

**Tech Stack:** Next.js 16 (App Router, Turbopack), React 19, TypeScript, Vitest + happy-dom, sharp.

## Global Constraints

- **Diploma360 output must not change.** After every phase, `npm run build` (default brand) and the diploma360 prerendered HTML for `/`, `/chi-siamo`, nav, footer must be identical to before. No new routes reachable on diploma360.
- **Prices VERBATIM** (both brands): `1.500 / 1.900 / 2.900 €`; rates `72,68 / 92,06 / 140,52 €/mese × 24`. **NEVER** "senza interessi".
- **Honest claims only:** never "facciamo al posto tuo"; "Diploma di Stato riconosciuto" (never MIM/MIUR); no AI-capability claims.
- **Luxottica is REMOVED** — exclude `partner/luxottica.png` from the Chi-siamo partner carousel even though the file exists.
- **Brand tokens only:** no hardcoded `Diploma360` / `www.diploma360.it` in `app/`,`components/`,`data/` (the guard test `test/no-hardcoded-brand.test.ts` enforces this). Use `brand.*`.
- **Design system unchanged:** reuse existing global classes (`.mainnav/.navitem/.mega/.navlink/.mega-link`, cards, sections); same palette. No CSS Modules for ported page markup — page-specific CSS is a plain global `app/<route>/<name>.css` imported at the top of the page.
- **`materiale/` is READ-ONLY.** The `consegna-valerio 2/*.dc.html` are 1:1 visual references to port, not to edit.
- **`npm test` output must stay pristine.**

```ts
// lib/assertBrand.ts
import { notFound } from 'next/navigation'
import { brand, type BrandId } from './brand'

/** Renders a 404 when the current build's brand is not `id`. Use at the top of
 *  brand-exclusive route components (e.g. /ripetizioni/*). */
export function assertBrand(id: BrandId): void {
  if (brand.id !== id) notFound()
}
```
